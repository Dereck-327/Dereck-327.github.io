## Getting started 🛠️

### 构建

```bash
git clone **
cd **

# 检查 node 和 npm 是否正常
node -v
npm -v

# 如果没有安装，用 nvm 装一个
nvm install --lts
nvm use --lts

## 安装hexo 脚手架
npm install -g hexo-cli

hexo s

hexo clean && hexo g && hexo d
```
