
# wsl2的快照与回滚

## 查看已安装的系统

```bash
wsl -l -v
```

```txt
  NAME              STATE           VERSION
* Ubuntu            Running         2
  docker-desktop    Stopped         2
```

## 做快照

```bash
wsl --export Ubuntu F:\WSL-bak\wsl-ubuntu.tar
```

## 回滚

### 注销当前系统

```bash
wsl --unregister Ubuntu
```

### 回滚

```bash
wsl --import D:\WSL\Ubuntu F:\WSL-bak\wsl-ubuntu.tar --version 2
```

```bash
# 安装的哪个版本的就写哪个版本的(比如我安装的是Ubuntu-24.04)
ubuntu2404 config --default-user USERNAME
```
